=====================================
Vassal Barons - Creation of Vassals
=====================================

■■■ DESCRIPTION ■■■
A Mount & Blade II: Bannerlord mod that enables:
- Offering vassalage to qualified NPCs through dialogue
- Selecting a fief (town or castle) for the new vassal
- Creating a custom clan name for the vassal
- Automatic inclusion of family members (spouse and children) in the new clan
- Native integration with the game's kingdom system

■■■ INSTALLATION ■■■
1. Extract the .zip file
2. Copy the "VassalBarons" folder to:
   "Steam\steamapps\common\Mount & Blade II Bannerlord\Modules"
3. Enable the mod in Bannerlord's launcher ("Mods" menu)

■■■ COMPATIBILITY ■■■
✔ Game Version: e1.2.12+ (tested up to e1.2.12)
✔ Compatible With:
   - Mods that don't alter lord dialogues
   - Kingdom systems (Kingdom Destruction, Diplomacy, etc.)
   - Mods Full Conversion
✖ Incompatible With:
   - Mods replacing "lord_talk_speak_diplomacy_2" (base dialogue)
