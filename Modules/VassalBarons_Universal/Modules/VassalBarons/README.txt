============================================
Vassal Barons - Criação de Vassalos Dinâmica
============================================

■■■ DESCRIÇÃO ■■■
Mod para Mount & Blade II: Bannerlord que permite:
- Oferecer vassalagem a NPCs qualificados via diálogo
- Selecionar um feudo (cidade ou castelo) para o novo vassalo
- Criar um nome personalizado para o clã do vassalo
- Automatizar a inclusão de familiares (cônjuge e filhos) no novo clã
- Integração nativa com o sistema de reinos do jogo

■■■ INSTALAÇÃO ■■■
1. Extraia o arquivo .zip
2. Copie a pasta "VassalBarons" para:
   "Steam\steamapps\common\Mount & Blade II Bannerlord\Modules"
3. Ative o mod no launcher do Bannerlord (menu "Mods")

■■■ COMPATIBILIDADE ■■■
✔ Versão do jogo: e1.2.12+ (testado até e1.2.12)
✔ Compatível com:
   - Mods que não alterem diálogos de lordes
   - Sistemas de reino (Kingdom Destruction, Diplomacy, etc.)
✖ Incompatível com:
   - Mods que substituam "lord_talk_speak_diplomacy_2" (diálogo base)
